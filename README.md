
## 基准测试实例说明

### 1. BenchmarkInstance_Delay
- **场景描述**：测试在无数据收集时延情况下的基准实例
- **适用场景**：适用于评估算法在理想时延条件下的性能表现
- **数据特点**：不包含数据收集时延因素，提供算法性能的理论上限参考

### 2. BenchmarkInstance_NoDelay
- **场景描述**：测试有确定的数据收集时延情况下的基准实例
- **对比内容**：包含PLONA算法与CPLEX求解器的对比数据
- **应用价值**：用于评估实际数据收集时延对算法性能的影响，比较启发式算法与精确求解方法的优劣

## 源代码说明

### scenarioConstruction/
该目录包含生成基准测试实例的完整实现代码：
- 支持自定义参数配置
- 可生成不同规模和复杂度的测试实例
- 包含实例验证逻辑，确保生成数据的有效性

### src/
该目录提供算法前期的参考实现：
- 包含PLONA算法的初步实现
- 提供数据处理的参考方法
- 可作为算法进一步开发和优化的基础框架
